# Stock-Market-Simulator-Project
This is a project simulating the outcome of various trading strategies using short and long averages on historical data. 

How to run: 
1. Download the repository
2. Run stockbotv2 in your browser.
3. Run 3dvisualiserare3 in your browser.
4. In stockbotv2 pick "choose file" and add US_test_data_2.
5. By pressing "utf√∂r test" you can generate a file that can be displayed in the 3d visualizer.
